## Installer nodejs et npm en local

# > cd backend
## installer les composants du projet.
```
npm install
```
### lancer le projet avec nodemon serve
```
npm start
```
Le serveur sera accessible sur http://localhost:3000

# > cd frontend
## installer les composants du projet.
```
npm install
```
### lancer le projet
```
npm run serve
```
Rendez-vous sur http://localhost:8080/
